#[FRONTEND] AWS - serverless hackathon 2016

### What you need
* Node.js and npm: https://nodejs.org/en/


### Installation
* Clone GIT repository
```bash
git clone git@github.com:SC5/aws-serverless-hackathon.git
```
* Install packages
```bash
npm install
```
* Now you ready to go!!!

### Run it
```bash
npm start
```
